# 🛡️ Cyber Attack Knowledge Hub (CyberHub)

**CyberHub** is a modern, production-quality educational platform designed to demystify cybersecurity for beginners. It provides a deep dive into cyber attack vectors, threat actor motivations, and enterprise-grade defense strategies through an immersive, dark-themed interface.

## ✨ Key Features

- **🚀 Interactive Attack Library**: 40+ curated real-world attack cases (Ransomware, Phishing, DDoS, etc.) with advanced filtering and search.
- **🌍 Live Threat Map**: A real-time, dynamic visualization of global cyber attacks using GeoJSON and Canvas.
- **🎣 Phishing Simulator**: Test your ability to spot malicious emails with an interactive learning game.
- **🤖 AI Security Assistant**: An integrated chat experience to answer complex security questions on the fly.
- **🔐 "Am I Pwned" Demo**: Check historical data breaches to see if your credentials have been exposed.
- **📊 Risk Rankings**: Stay ahead of the curve with our 3-5 year security outlook and impact analysis.
- **📱 Premium UX**: Fully responsive, glassmorphic design with custom-built command palettes (Ctrl+K) and smooth animations.

## 🛠️ Tech Stack

- **Frontend**: Pure HTML5, Vanilla CSS3 (Custom Design System), JavaScript (ES6+).
- **Visualization**: Canvas API for the Global Map and background effects.
- **Deployment**: Configured for Vercel, Render, and Node.js.
